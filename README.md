# pequena-loja-virtual
## sobre:
Uma pequena loja virtual com sistema de pagamento conectado com Stripe. Todos os produtos são listados da stripe e gerenciados no banco de dados da aplicação. Essa é uma aplicação fullstack, tanto o front quanto backend foram feitos em Next.js.
### A aplicação conta com:
<ul>
  <li>Sistema de login via Google</li>
  <li>Carrinho de compras, com alterações sincronizadas ao banco de dados</li>
  <li>Filtragem de produtos por nome, categoria e tamanhos</li>
  <li>Sistema de checkout para finalização de compras.</li>
</ul>
